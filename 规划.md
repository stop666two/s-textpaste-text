# S-TextPaste 规划文档

## 项目定义

绝密级零信任端到端加密文本分享服务。所有加解密在浏览器完成，服务端仅存储密文。

- **仓库**: `https://github.com/stop666two/S-TextPaste`
- **部署**: Cloudflare Workers + D1，一键按钮部署
- **本地**: React 18 + Vite + Node.js

## 加密方案

### 密码派生密钥 (512-bit KDF)

```
salt = random(32字节)
H1   = SHA-256(password || salt)    32字节
H2   = MD5(password || salt)        16字节
H3   = SHA-256(salt || password)    32字节, 取前16

K_512 = interleave(H1, H2) || H1[16..31] || H3[0..15]  = 64字节

然后链式迭代 12 轮，每轮 SHA-256⊕MD5 XOR 交织
```

前 32 字节用作 AES-256 密钥。

### 后量子密钥 A (1024-bit KDF-A, 24 轮)

```
subSalt = SHA-256(salt || "PQ-A-v3")
以 SHA-256 为主导，MD5 嵌套链式迭代 24 轮
输出 128 字节
```

### 后量子密钥 B (1024-bit KDF-B, 24 轮)

```
subSalt = SHA-256(salt || "PQ-B-v3")
以 MD5 为主导，SHA-256 嵌套链式迭代 24 轮 (与 A 完全逆向)
输出 128 字节
```

### HMAC 密钥 (256-bit, 7 轮)

```
subSalt = SHA-256(salt || "H-v3")
独立链式迭代 7 轮，SHA-256(md5(chain||pw)||chain||salt||round)
前 32 字节用作 HMAC-SHA-256 密钥
```

### 三重包络加密

```
明文
  → DEK(随机32字节) ──AES-256-GCM──→ CT1
  → KeyA(1024bit)    ──AES-256-GCM──→ CT2
  → KeyB(1024bit)    ──AES-256-GCM──→ CT3 = encrypted_payload
  → HMAC-SHA-256(HMACkey, 全载荷字段) = 防篡改标签
```

### 对称密钥模式

用户提供密钥 → AES-256 直接加密明文 → SHA-256+MD5 推导 PQ 密钥 → 三重包络

### 非对称模式

RSA-OAEP(2048) 公钥加密 DEK → AES-256-GCM 加密明文

## 数据库 (Cloudflare D1)

```sql
CREATE TABLE IF NOT EXISTS pastes (
  id TEXT PRIMARY KEY,
  mode TEXT NOT NULL,               -- password | symmetric | asymmetric
  salt TEXT,                        -- Base64 salt
  encrypted_payload TEXT NOT NULL,  -- 完整加密信封
  hint TEXT DEFAULT '',             -- 密码提示
  delete_token_hash TEXT NOT NULL,  -- SHA-256 哈希
  expires_at INTEGER,               -- Unix 时间戳
  max_views INTEGER DEFAULT -1,     -- -1 无限
  view_count INTEGER DEFAULT 0,
  burn_after_read INTEGER DEFAULT 0,-- 0 或 1
  created_at INTEGER NOT NULL,
  pubkey_fingerprint TEXT           -- 非对称模式公钥指纹
);
```

## API

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/paste` | 创建粘贴 |
| GET | `/api/paste/:id` | 获取粘贴 (仅返回密文+生命周期) |
| POST | `/api/paste/:id/view` | 记录查看 + 触发阅后即焚 |
| DELETE | `/api/paste/:id` | 删除粘贴 (需 X-Delete-Token) |

## 前端页面

### CreatePage
- CodeMirror 6 编辑器 + 实时 Markdown 预览
- 三种加密模式: 密码 / 对称密钥 / 非对称
- 后量子开关 (默认开启)
- 生命周期: 过期时间 / 最大浏览次数 / 阅后即焚
- 自定义短链接 (4-64 字符)
- 创建成功展示链接 + 删除令牌

### ReadPage
- 安全性仪表盘 (加密算法 / 密钥强度 / 量子安全)
- 密码提示
- 根据模式动态显示输入项
- 5 次错误锁定
- 解密成功跳转 ViewPage (state 内存传递)

### ViewPage
- Markdown 渲染: GFM + Mermaid + KaTeX + 代码高亮
- 安全性仪表盘
- 删除面板 (输入令牌)
- 内容仅在 React Router state 中，URL 无泄露

### 通用
- 暗黑模式 (自动检测 + localStorage)
- 中/英文切换
- 首次访问免责声明 (localStorage 记录)
- 标题可点击回主页

## 安全

| 机制 | 实现 |
|------|------|
| 加密 | AES-256-GCM 认证加密 |
| 完整性 | HMAC-SHA-256 全载荷 |
| 传输 | HTTPS (Cloudflare) |
| CSP | `default-src 'self'` |
| ID | 32 字符随机 (64^32) 无偏分布 |
| 速率 | 30次/分钟/IP |
| 令牌 | SHA-256 哈希存储 |
| 内容 | React Router state，不出现在 URL |
| 元数据 | 全部嵌入密文载荷 |
